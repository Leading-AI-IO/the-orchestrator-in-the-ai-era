## 📖 概要

1つの専門性を深めることがキャリアの正解だった時代は終わった。

AIは実行を奪い、単一専門性の価値を急速に圧縮している。マサチューセッツ工科大学（MIT）の研究は生成AIによる生産性40%向上を示し、ハーバード・ビジネス・スクールはAIの能力境界が「ギザギザ（Jagged Frontier）」であることを明らかにした。その凹凸を読み解くには、複数の専門性からの視点が必要になる。

AI時代に最も希少なのは、Business・Technology・Creativeの3つの専門性をプロレベルで理解し、人とAIの両方をオーケストレーションできる人材だ。

**それがオーケストレーターであり、まだ誰も定義していない。**

本書は、越境によって獲得した「眼（複眼性）」と、AI時代の「武器（思考のOS）」を統合した新しい人材像——オーケストレーター——を、世界で初めて定義する。

## 🏗️ 構成

本書は全6章で構成される。

| 章 | タイトル | 内容 |
|---|---|---|
| 第1章 | 3つの世界、3つの言語 | Business・Technology・Creativeの3領域が異なる思考回路で動いている構造。領域間の断絶と、その断絶が生むビジネスコスト |
| 第2章 | なぜ単一の専門性はAI時代に脆弱なのか | アダム・スミスの分業原理から250年。AIが「実行」を圧縮し、kind環境が崩壊し、単一専門性の価値が構造的に毀損される現実 |
| 第3章 | 越境という行為の本質 | 田川欣哉氏の「複眼性」と「振り子の思考」、濱口秀司氏の「バイアス・ブレイク」と「ストラクチャード・ケイオス」。越境が器用貧乏ではなく、解像度の掛け算であることの論証 |
| 第4章 | 思考のOS — AIを操る者が身につけるべき武器 | 10:80:10の法則、クリティカルシンキング、コンテキストエンジニアリング、1次情報×LLM仮説検証サイクル、4つのプロンプト・レバー。Depth & Velocityの方法論体系から抽出した5つの武器 |
| 第5章 | 眼と武器の統合 — オーケストレーターの誕生 | 複眼性と思考のOSが1人の人間の中で統合された時に何が起きるか。加算ではなく乗算。濱口秀司氏をAI以前の最高到達点として位置づけ、AI時代への更新を論じる |
| 第6章 | オーケストレーターの定義 | まだ誰も到達していない人材像の定義。3つの眼の精緻な記述、思考のOS体系、到達への5段階の実践論 |
